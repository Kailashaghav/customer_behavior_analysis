# customer_behavior_analysis
data analytics project showcasting customer behavior analysis using python and sql and power BI
📊 Data Analytics Project

📌 Overview

This project demonstrates an end-to-end data analytics workflow, including data loading, cleaning, exploration, querying, visualization, and reporting. The goal is to extract meaningful insights from raw data and present them in an interactive and business-friendly format.

⸻

📂 Dataset
	•	Source:customer_shopping_behavior_csv
	•	Format: CSV / Excel / Database
	•	Description:
The dataset contains information related to (e.g., customers, sales, products, etc.) used to analyze trends, patterns, and performance metrics.

⸻

🛠️ Tools & Technologies
	•	Python (Pandas, NumPy, Matplotlib, Seaborn)
	•	SQL (PostgreSQL / MySQL / SQL Server)
	•	Power BI (Dashboard creation & visualization)
	•	Jupyter Notebook / VS Code
	•	Gamma (for PPT creation)

⸻

🔄 Project Workflow

1.⁠ ⁠Data Loading
	•	Imported dataset using Python (Pandas)
	•	Verified structure, columns, and data types

2.⁠ ⁠Data Cleaning
	•	Handled missing values
	•	Removed duplicates
	•	Fixed inconsistent formats
	•	Converted data types where necessary

3.⁠ ⁠Exploratory Data Analysis (EDA)
	•	Summary statistics
	•	Data distribution analysis
	•	Correlation analysis
	•	Visualization using charts and graphs

4.⁠ ⁠SQL Analysis
	•	Connected dataset to SQL database
	•	Executed queries such as:
	•	Aggregations (SUM, AVG)
	•	Filtering and sorting
	•	Subqueries
	•	Grouping and joins

5.⁠ ⁠Dashboard Creation
	•	Built interactive dashboard in Power BI
	•	Key features:
	•	KPIs and metrics
	•	Filters and slicers
	•	Trend analysis
	•	Visual storytelling

6.⁠ ⁠Reporting
	•	Generated insights report
	•	Highlighted key findings and business recommendations

7.⁠ ⁠Presentation (PPT)
	•	Created professional presentation using Gamma
	•	Included:
	•	Problem statement
	•	Methodology
	•	Visual insights
	•	Conclusion
